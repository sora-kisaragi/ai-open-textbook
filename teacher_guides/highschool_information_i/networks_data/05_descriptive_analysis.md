# 教師用ガイド: 記述統計によるデータ分析

## レッスンのねらい

問いとデータ型から要約を選び、度数、割合、平均値、中央値、範囲、教材で定義した四分位範囲を計算します。Pythonは手順の再現に使い、一つの統計量から母集団や因果を断定させません。

## 前提確認

D5の欠測と型、割合と四則演算を確認します。四分位数の計算法が複数あることを明示し、本教材の手順を固定します。

## 学習者用本文との対応

1. 問いに合う要約を選ぶ。（D6.O1）
2. 8個の合成値で度数、割合、中心、散らばりを計算する。（D6.O2）
3. 平均と中央値、同平均で異なる分布を比較する。（D6.O3）
4. Python結果を手計算と照合する。

## 時間配分の目安

70〜110分です。第1時に選択と計算、第2時に分布、外れ値、解釈の限界を扱います。

## 進め方と形成的評価

D6.O1は選択理由、D6.O2は式・単位・分母、D6.O3は同じ平均を持つ二つの分布の比較で確認します。計算機利用時も、欠測処理と並べ替えを説明させます。

## 想定される誤解

- すべての数値へ平均を使う。
- 割合の分母を書かない。
- 平均値を典型・散らばり・原因のすべてとする。
- 8回を全期間へ一般化する。

## 合成データと出典上の注意

利用人数は計算検証用の合成データです。統計的推測や実在施設の評価は扱いません。
