# Copyright Policy

## Basic rule

Create original educational content.
Do not copy existing textbooks, problem books, diagrams, answer keys, or paid materials.

## Sources

Use sources for facts, curriculum references, legal context, and technical definitions.
Prefer primary sources.

## Quotation

Use quotation only when necessary.
Keep it short, attribute it, and explain why it is needed.

## AI-generated content

AI-generated content still needs originality review.
AI output may reproduce protected patterns, so reviewers must check for suspicious similarity.

## Data records

Use `source_refs` for records that depend on external sources.
Use `review_status` to flag unresolved copyright questions.
