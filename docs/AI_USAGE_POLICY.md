# AI Usage Policy

## Allowed uses

- Drafting outlines
- Drafting lessons
- Generating exercises
- Generating rubrics
- Building validation scripts
- Preparing review checklists
- Summarizing review findings

## Prohibited uses

- Publishing without human review
- Inventing sources
- Hiding uncertainty
- Storing private chain-of-thought
- Using student personal data without explicit approval

## Disclosure

Pull requests should include an AI usage summary:

- Model used
- Task performed
- Human checks completed
- Remaining uncertainty

## Related draft policy

For draft AI-assisted writing quality guidance, see
