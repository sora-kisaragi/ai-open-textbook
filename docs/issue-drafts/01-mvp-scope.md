## Purpose

Define the MVP boundary for Information I / Programming Basics as an open educational material project. This is not an official government-approved textbook.

## Scope

- Subject: Japanese High School Information I
- Unit: Programming Basics
- Lessons: variables and assignment, conditionals, loops

## Acceptance criteria

- [ ] MVP includes only variables, conditionals, and loops.
- [ ] The project description avoids any official textbook claim (use: open educational material, supplemental resource, self-study material, teacher support material).
- [ ] Human review gates are listed (per `AGENTS.md`: release, curriculum alignment claims, copyright-risk decisions, age-appropriateness, `approved` -> `published`).
- [ ] Out-of-scope items are explicit (e.g., functions, lists/arrays, other Information I units, localization).
- [ ] Dependencies between curriculum, lessons, data, review, and release work are documented.

## Review gates

- Human review required before scope is treated as fixed.
