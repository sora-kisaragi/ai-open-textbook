## Clean-Room Rule

The project may compare whether a public offering describes a broad instructional
feature. It must not retain or imitate publisher wording, examples, datasets,
exercise logic, diagrams, page layouts, teacher guidance, answer material, or
other expressive choices. No publisher page is an educational source record.
The register below retains only the public page identity, access date, format
facts, and broad feature categories needed to make the comparison auditable.
No quotation, screenshot, sample page, or feature implementation detail is
retained for authoring.

## Feature Baseline

Human review of the integration baseline found a coherent scope and delivery
route, but inconsistent depth for a textbook-like classroom and self-study
experience. The following comparison is limited to feature presence.

| Non-expressive feature category | Integration-baseline observation | Issue #91 pilot target |
| --- | --- | --- |
| Explanations and concept models | Present, but not consistently deep or explicit across all lessons. | Language- and tool-independent model with inspectable reasoning. |
| Visual models | Not consistently available with offline and non-visual equivalents. | Original offline figure where meaningful, specific alt text, and complete text/data equivalent. |
| Worked examples and contrasts | Present unevenly; reasoning and contrasting errors are not consistently exposed. | Reasoned worked example plus misconception contrast where relevant. |
| Guided activity and feedback | Not consistently staged from support to independent work. | Guided attempt, misconception-targeted feedback, and independent transfer. |
| Readiness and remediation | Routes exist unevenly. | Short readiness check, explicit remediation, retry, and help decision. |
| Self-study support | Explanations exist, but staged hints, success criteria, and mastery decisions are not consistently available. | Two staged hints, full explanation, learner-facing criteria, retry route, and mastery decision. |
| Teacher operation | Timing support exists, but materials, expected responses, feedback moves, accessibility fallback, stopping point, and extension are uneven. | Complete teacher-operable evidence for each mandatory period. |
| Cumulative practice | Lesson-local practice exists; cumulative and interleaved practice is not yet consistent. | Later unit work must revisit prior objectives in changed contexts. |
| Claim review | Source records exist; claim-level review evidence remains uneven for externally checkable claims. | Exact review evidence or explicit open questions, separate from this comparison. |

## Interpretation

The table is a planning baseline, not a quality ranking and not evidence that any
public product or this project is approved, equivalent, complete, or suitable for
release. The four pilots test whether the depth contract is reviewable before the
same architecture is scaled to Units A through D.
