# 記述統計によるデータ分析

## 学習目標

1. データ型と分析の問いに合う要約を選べる。（D6.O1）
2. 小さなデータの度数、割合、中心、散らばりを計算し解釈できる。（D6.O2）
3. 分布や外れ値によって、一つの要約が誤解を招く理由を説明できる。（D6.O3）

## 前提知識

- D5で学んだ、表形式データ、型、欠測、クリーニング履歴。
- 割合、四則演算、大小関係。必要な計算は本文で確認します。

## 導入: 一つの数にまとめる前に問いを決める

クリーニング済みの合成データとして、8回の観測で得た図書室の利用人数を使います。

`12, 14, 15, 15, 16, 18, 19, 31`

「典型的な人数」「20人以上の回の割合」「ばらつき」では、必要な要約が異なります。

## データ型と問いに合う要約

| データと問い | 適した要約の例 |
| --- | --- |
| 曜日ごとの観測回数 | 度数、割合 |
| 利用人数の典型的な大きさ | 平均値、中央値 |
| 利用人数の散らばり | 範囲、四分位範囲など |
| 蔵書分類の最多カテゴリ | 最頻値、度数表 |

識別番号の平均には意味がありません。数で書かれていても、大小や差に意味があるかを確認します。

## 度数と割合

20人以上は `31` の1回です。

- 度数: 1回
- 全観測数: 8回
- 割合: `1 / 8 = 0.125`、つまり12.5%

割合だけを書くと、母数が分かりません。「8回中1回（12.5%）」のように、度数と分母も示します。

## 平均値と中央値

平均値は、すべての値を足して個数で割ります。

`(12 + 14 + 15 + 15 + 16 + 18 + 19 + 31) / 8 = 17.5`

中央値は、小さい順に並べた中央です。8個なので4番目と5番目の平均を取ります。

`(15 + 16) / 2 = 15.5`

平均値17.5人と中央値15.5人の差は、31人という大きな値の影響を示します。どちらが「正しい」のではなく、何を表したいかで選びます。

## 散らばりを見る

最小値は12、最大値は31なので、範囲は `31 - 12 = 19` です。範囲は簡単ですが、最小値と最大値だけで決まります。

データを半分に分けます。

- 下半分: `12, 14, 15, 15`。中央は `(14 + 15) / 2 = 14.5`
- 上半分: `16, 18, 19, 31`。中央は `(18 + 19) / 2 = 18.5`

この教材の手順では、四分位範囲を `18.5 - 14.5 = 4` とします。四分位数には複数の計算法があるため、方法を明記して比較します。

## Pythonで計算手順を再現する

Pythonは計算を繰り返す道具です。何を要約しているかは人が決めます。

```python
people = [12, 14, 15, 15, 16, 18, 19, 31]

total = 0
for value in people:
    total = total + value

mean = total / len(people)
middle = (people[3] + people[4]) / 2
data_range = people[-1] - people[0]

print(mean)
print(middle)
print(data_range)
```

並びが保証されていないデータで中央値を求めるなら、先に並べ替える必要があります。欠測を0として混ぜてはいけません。

## 一つの要約だけでは見えないこと

次の二つはどちらも平均15人です。

- A: `14, 15, 15, 16`
- B: `0, 10, 20, 30`

しかし散らばりは大きく違います。また、観測データだけでは「座席を増やせば学習成果が上がる」といった因果関係は確定できません。記述統計は、観測したデータの特徴を要約するものであり、母集団全体や原因を自動的に証明するものではありません。

## よくある間違い

- 識別番号など、差に意味がない数値の平均を求める。
- 割合の分母を示さない。
- 欠測を0として平均へ入れる。
- 平均値だけで典型と散らばりを説明する。
- 観測した8回の結果を、すべての日へそのまま一般化する。

## 自己確認の診断と修正

- すぐ平均を求めた場合は、データ型と分析の問いへ戻ります。
- 数値だけを書いた場合は、単位、分母、対象期間を付けます。
- 一般化した場合は、標本の範囲、欠測、偏り、観測数を確認します。

## まとめ

- 要約は、データ型と分析の問いに合わせて選びます。
- 度数と割合、平均値と中央値、中心と散らばりを組み合わせて解釈します。
- 外れ値や分布、欠測、標本範囲を示し、一つの統計量で断定しません。

## 練習

- **問いに合う要約を選ぶ**
- **度数、割合、中心、散らばりを計算する**
- **平均だけでは見えない分布を説明する**
- **合成データの分析結論を評価する**
