# 変数と代入

## 学習目標

このレッスンのあと、学習者は次のことができるようになります。

1. 変数がなぜ役に立つのかを説明できる。
2. Pythonで変数に値を代入できる。
3. 簡単な代入のあとで、変数の値を予測できる。

## 前提知識

- コンピュータプログラムは順番に実行される命令である、という基本的な考え方。
- 数字を使った簡単な計算は、このレッスンの中で確認します。

## 導入

プログラムでは、値をあとで使うために覚えておきたい場面がよくあります。
変数は値に名前を付けるしくみです。名前を付けておくと、プログラムはその値をあとから使えます。

変数に値を入れることを代入といいます。Pythonでは、`score = 80` のように書くと、変数 `score` に値 `80` を代入します。

代入した値を画面で確認したいときは、`print` 関数を使います。`print(score)` と書くと、変数 `score` の現在の値が表示されます。

## 例

```python
score = 80
print(score)
```

変数 `score` には値 `80` が保存されます。
そのあとで `print(score)` が実行されるので、保存されている値 `80` が表示されます。

## よくある間違い

### 間違い: 代入を等しいこととして読む

```python
x = x + 1
```

数学の式として見ると、これは成り立たない等式に見えます。
Pythonでは、「現在の `x + 1` の値を計算し、その結果をもう一度 `x` に保存する」という意味です。

### 間違い: 代入する前に変数を使う

```python
print(total)
```

それより前に `total` へ値が代入されていない場合、このコードは失敗します。

## 自己確認

- `x = 3` のあとに `x = x + 2` を実行すると、`x` の値はいくつになりますか。
- `score = 80` を実行したあと、`print(score)` では何が表示されますか。

## まとめ

- 変数は、値に名前を付けてあとから使うためのしくみです。
- 代入は、変数に値を保存したり、変数の値を更新したりする操作です。
- `print` 関数を使うと、変数の現在の値を確認できます。
