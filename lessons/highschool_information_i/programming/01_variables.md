# 変数と代入

## 学習目標

このレッスンのあと、学習者は次のことができるようになります。

1. 変数がなぜ役に立つのかを説明できる。
2. Pythonで変数に値を代入できる。
3. 簡単な代入のあとで、変数の値を予測できる。

## 前提知識

- 数字を使った簡単な計算は、このレッスンの中で確認します。

上から順に実行する考え方は、このレッスンで初めて扱い、ここで確認します。すでに知っている必要はありません。

## 診断とミニレッスン: 命令の順序

まず、次の三つの命令を上から順に読んでください。

```text
1. 箱に6を入れる
2. 箱の値に4を足し、結果を箱に入れ直す
3. 箱の現在の値を記録する
```

| 命令のあと | 箱の値 |
| --- | ---: |
| 1 | 6 |
| 2 | 10 |
| 3 | 10 |

3番目の記録は、その時点の値10です。もし6と答えた場合は、2番目の命令による更新を飛ばしていないか確認します。プログラムも、特別な指示がない限り、上の行から順に一行ずつ実行します。

## 導入

プログラムでは、値をあとで使うために覚えておきたい場面がよくあります。
変数は値に名前を付けるしくみです。名前を付けておくと、プログラムはその値をあとから使えます。

変数に値を入れることを代入といいます。Pythonでは、`score = 80` のように書くと、名前 `score` と値 `80` を対応付けます。このレッスンでは、この対応付けを初心者向けに「値を入れる」「保存する」と表現します。

代入した値を画面で確認したいときは、`print` 関数を使います。`print(score)` と書くと、変数 `score` の現在の値が表示されます。

## 例

```python
score = 80
print(score)
```

名前 `score` は値 `80` と対応付けられます。
そのあとで `print(score)` が実行されるので、`score` に対応する値 `80` が表示されます。

### 印刷待ちの状態を表す

星見高校メディアラボに、1部だけ印刷する6ページの依頼があり、新しく1部4ページの依頼が届いた場面を考えます。
この例ではページ数と枚数が同じです。まず表で追うと、「6を対応付ける」「4増やす」「現在値を確認する」という順序です。
依頼IDや数値は合成データであり、実在する生徒名、文書内容、利用履歴へ置き換えません。

```python
pages_waiting = 6
pages_waiting = pages_waiting + 4
print(pages_waiting)
```

`pages_waiting` という名前は、その値が何を表すかを読み手に伝えます。Pythonの書き方は具体例であり、本質は、名前を付けた値を順番に更新して状態を表すことです。

## よくある間違い

### 間違い: 代入を等しいこととして読む

```python
x = x + 1
```

数学の式として見ると、これは成り立たない等式に見えます。
Pythonでは、「現在の `x + 1` の値を計算し、その結果をもう一度 `x` へ対応付ける」という意味です。

### 間違い: 代入する前に変数を使う

```python
print(total)
```

それより前に `total` へ値が代入されていない場合、このコードは失敗します。

## 自己確認

- `x = 3` のあとに `x = x + 2` を実行すると、`x` の値はいくつになりますか。
- `score = 80` を実行したあと、`print(score)` では何が表示されますか。
- `pages_waiting` のように、値の意味が分かる名前を付ける利点は何ですか。

## まとめ

- 変数は、値に名前を付けてあとから使うためのしくみです。
- 代入は、名前と値を対応付けたり、その対応を更新したりする操作です。
- `print` 関数を使うと、変数の現在の値を確認できます。
