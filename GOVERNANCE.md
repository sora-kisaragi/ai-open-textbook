# Governance

## Roles

- Maintainer: approves releases, licenses, and governance changes.
- Reviewer: checks educational quality, accuracy, accessibility, and sources.
- Contributor: proposes content, data, and code changes.
- Agent: assists with drafting, validation, and review preparation.

## Decision rule

Routine changes require one human reviewer.
Release changes require maintainer approval.
License, governance, and data model changes require an explicit governance decision record.

## Decision records

Use `docs/ADR/` for durable decisions.
